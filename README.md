# ZeiterFree
## ¿Qué es Zeiter Free?
Zeiter free es una aplicación web de control de accesos
##Desarrollo
Esta aplicación esta desarrollada utilizando NodeJS y Express utilizando una base de datos NoSQL (MongoDB).
